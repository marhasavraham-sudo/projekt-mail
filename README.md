# AutoScheduler AI

מערכת מבוססת AI לניהול יומן אוטומטי המייעלת את תהליך אישור הפגישות ביומן.

## רציונאל
הצפת הודעות מייל המכילות זימונים לפגישות גוזלת זמן רב. המטרה של פרויקט זה היא לבנות סוכן (Agent) שקורא את המיילים הנכנסים, מנתח את התוכן באמצעות מודל שפה (LLM) כדי לחלץ פרטים (תאריך, שעה, מיקום, נושא), בודק זמינות ביומן Google האישי, ומבצע פעולה:
1. במידה והלו"ז פנוי - מוסיף את הפגישה ליומן.
2. במידה והלו"ז תפוס - שולח מייל חוזר למוען המודיע על חוסר זמינות.

## טכנולוגיות
- **Python**: שפת הפיתוח.
- **Gmail API & Google Calendar API**: לתקשורת עם סביבת העבודה של גוגל.
- **OpenAI API / Gemini API**: לניתוח הטקסט (NLP) וחילוץ נתונים מהמייל.
- **OAuth2**: לניהול הרשאות מאובטח.
